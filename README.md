# Hack-Hunt

A multiplayer hacking simulator game.
Inspired by the game "Hacknet".

## Getting Started

```
git clone
cd hack-hunt
npm install
npm run dev
```

## Built With

- [Svelte](https://svelte.dev/) - The web framework used
